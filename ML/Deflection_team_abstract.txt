As the final model for ML-CUP, we selected a fully-connected feedforward neural network (Multi-Layer Perceptron) trained using a stepsize-restricted deflected subgradient learning algorithm through model selection. The architecture consists of an input layer with 12 units, two hidden layers (with the number of neurons selected via model selection), and an output layer with 4 units. The hidden layers employ Leaky ReLU activation functions, while the output layer uses a linear activation function. The hidden layers were initialized using He (Kaiming) initialization, based on the fan-in criterion, whereas the output layer was initialized with Xavier initialization. L1 regularization was applied, combined with early stopping based on validation performance. For model validation, we first split the data to define an internal test set, and then performed 5-fold cross-validation to obtain training and validation sets.
